# @visual-json/ui-shared

> **Internal package** — not intended for direct consumption.

This package contains shared UI utilities used by `@visual-json/react` and `@visual-json/vue`. All exports are internal implementation details and **not public APIs** — they may change, move, or be removed at any time without notice.

**Please do not depend on this package directly.** Use `@visual-json/react` or `@visual-json/vue` instead.
